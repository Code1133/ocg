# MapPreset

MapPreset is Data asset that is core to level creation.
You can set various properties in this asset to create level you want.

## Editing MapPreset
OCG Window 또는 MapPreset 커스텀 에디터에서 MapPreset 에셋을 편집하고, 레벨을 생성할 수 있습니다.
