# Limitations

One-Click Level Generator는 쉽고 빠른 레벨 생성을 목표로 하지만, 몇 가지 제한 사항이 있습니다.

- 일부 기능은 아직 개발 중입니다.
- 레벨 생성 속도가 느릴 수 있습니다.

추가로 버그나 문제가 발생할 경우, [GitHub Issues]에 보고해 주세요.

[GitHub Issues]: https://github.com/Code1133/ocg-docs/issues
